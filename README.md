# 🚀 SpaceBlaster

A classic 2D space shooter game built with **Python** and **Pygame**. Defend the galaxy by blasting enemy ships before they reach you!

---

## 🎮 Gameplay

- Move your spaceship **left and right** to dodge enemies.
- Press **SPACE** to shoot bullets at incoming enemies.
- Each enemy hit earns you **+1 point**.
- If an enemy reaches your ship — **Game Over!**

---

## 📸 Preview

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed. Then install Pygame:

```bash
pip install pygame
```

### Run the Game

```bash
python game.py
```

---

## 🕹️ Controls

| Key         | Action          |
|-------------|-----------------|
| `←` Left    | Move left       |
| `→` Right   | Move right      |
| `SPACE`     | Shoot bullet    |
| Close window | Quit game      |
